# RCCL

[RCCL](https://rocmdocs.amd.com/projects/rccl/en/latest/) is an optimized inter-GPU communication library for AMD GPUs.
It provides equivalent functionality to [NCCL][ref-communication-nccl] for AMD GPUs.
